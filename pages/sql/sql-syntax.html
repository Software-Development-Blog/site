SQL SYNTAX

DATA TYPES. Boolean, Int, Bigint, Decimal, Float, Varchar, Text, Date, Time, Timestamp, Blob.

CONSTRAINTS. Unique, Primary Key, Foreign Key (Many-to-One).

DATA DEFINITION LANGUAGE (DDL). Create Table, Alter Table, Drop Table, Create Index, Drop Index.

DATA QUERY LANGUAGE (DQL). Select, Union, Intersect, Difference, Inner Join, Outer Join (Left Join, Right Join).




DATA MANIPULATION LANGUAGE (DML). Insert, Update, Delete.

DATA CONTROL LANGUAGE (DCL). Grant, Revoke.

TRANSACTION CONTROL LANGUAGE (TCL). Commit, Rollback.

DQL EXAMPLE
SELECT column1, column2
FROM table_name
WHERE condition
ORDER BY column1 ASC, column2 DESC
LIMIT 10;

Suppose there are two tables, employees and departments. To retrieve a list of employees along with their associated department information:

SELECT employees.employee_id, employees.employee_name, departments.department_name
FROM employees
INNER JOIN departments ON employees.department_id = departments.department_id;

To retrieve a list of all employees, including those who may not be assigned to any department:

SELECT employees.employee_id, employees.employee_name, departments.department_name
FROM employees
LEFT JOIN departments ON employees.department_id = departments.department_id;

In this query, the LEFT JOIN returns all rows from the employees table and the matching rows from the departments table. If there is no matching department for an employee, the department_name will be NULL.
To retrieve a list of all departments, including those without any employees, use a RIGHT JOIN:

SELECT employees.employee_id, employees.employee_name, departments.department_name
FROM employees
RIGHT JOIN departments ON employees.department_id = departments.department_id;

In this query, the RIGHT JOIN returns all rows from the departments table and the matching rows from the employees table. If there are departments without any assigned employees, the employee_id and employee_name will be NULL for those rows.

DML EXAMPLE
INSERT INTO employees (employee_id, employee_name, department_id)
VALUES (1, 'John Smith', 101);

UPDATE employees
SET department_id = 102
WHERE employee_id = 1;

DELETE FROM employees
WHERE employee_id = 3;

INDEXES. Indexes are commonly used on columns that are frequently used in WHERE clauses for filtering, JOIN operations, and ORDER BY clauses for sorting.

CREATE INDEX index_name ON table_name (column1, column2);

ALTER TABLE table_name
RENAME INDEX old_index_name TO new_index_name;

DROP INDEX index_name ON table_name;

INSERTION SORT

Insertion Sort is a sorting algorithm with a complexity of O(n^2).

Idea of the algorithm is that we assume that the left subarray (containing some number of elements) of the given array is already sorted. At the beginning this subarray consists of one (first) element of the given array. We take the next (right) element to the sorted subarray and compare it with the elements from the sorted subarray. If needed, we rearrange the sorted subarray. So at each iteration we will have enlarged by one element sorted subarray until length of the subarray is not equal to a length of the given array.

Letâ€™s see an example. Given a sequence [5], 4, 3, 2, 1. [5] assumed sorted subarray, 4 is the next element to compare with others from the left subarray. 4 < 5. So, we rearrange the sorted subarray and enlarge it with 4: [4, 5], 3, 2, 1. Next element to compare is 3: [4, 5], 3, 2, 1. In a similar manner we rearrange sorted subarray and enlarge it by 3: [3, 4, 5], 2, 1. Then: [2, 3, 4, 5], 1. And, finally, [1, 2, 3, 4, 5].

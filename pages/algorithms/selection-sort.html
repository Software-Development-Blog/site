SELECTION SORT

Selection sort is a sorting algorithm with a complexity of O(n^2).

Idea of the algorithm is that we take a given array, select its minimum element and place it in the first position. So the most left subarray with one element (which we found as a minimum one) is already sorted. Next we take a given array from its next (second) element to the end, find a minimum element, and place it on the second position. So, now we have a subarray of two elements which are sorted. We continue in a similar manner until all elements are sorted.

Letâ€™s see an example. Given a sequence 5, 4, 3, 2, 1. 1 is a minimum element and we place it on the first position instead of the value of 5 (and value of 5 is now placed on the place of value 1): [1], 4, 3, 2, 5. So, subarray [1] is already sorted. Next minimum element is a value of 2. So, we place it on a second position instead of the value of 4, and the value of 4 (second position) is placed on the place of value 2: [1, 2], 3, 4, 5. Next minimum element is a value of 3. There is no need to replace it with other elements: [1, 2, 3], 4, 5. Similarly [1, 2, 3, 4], 5. And, finally, [1, 2, 3, 4, 5].

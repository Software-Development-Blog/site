Bubble Sort

Insertion Sort is a sorting algorithm with a complexity of O(n^2).

Idea of the algorithm is that within sorting actions we move a maximum element from a given array to the most right position - a maximum element “pops up” like buble. Then we look through remaining elements (except already sorted) and within sorting actions move the maximum to the place before the previous maximum element (on a pre-last position). We repeat these operations until all elements will be placed on their positions. Sorting action is an operation of comparing two adjacent elements and, if they are not ordered, - swapping them.

Let’s see an example. Given a sequence 5, 4, 3, 2, 1. Maximum element is 5, and we have to move it on the most right position by comparing and swapping all the elements starting from the first one:
4, 5, 3, 2, 1
4, 3, 5, 2, 1
4, 3, 2, 5, 1
4, 3, 2, 1, [5]

Now, [5] is a sorted subarray. We continue compare and swap unordered elements:
3, 4, 2, 1, [5]
3, 2, 4, 1, [5]
3, 2, 1, [4, 5]

Continue in similar manner:
2, 3, 1, [4, 5]
2, 1, [3, 4, 5]

Finally:
1, [2, 3, 4, 5] and [1, 2, 3, 4, 5]
